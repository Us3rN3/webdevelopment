de text staat horizontaal gecenterd, omdat de div op text-align: center; staat.

dit komt omdat vertical-align: center; niet bestaat je kan wel middle gebruiken.

Vertical align zorgt ervoor hoe de div staat verticaal gezien, bij de grote divs kan je dit niet zo goed zien maar bij de kleine wel deze staan dan mooi in het midden.

.verticaal-gecentreerd { height: 100px; line-height: 100px; }

.container { display: table; height: 200px; } .verticaal-gecentreerd { display: table-cell; vertical-align: middle; }

omdat tall als laatste wordt gedefinieerd zal dit voorrang hebben op class.
